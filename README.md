# Luontopeli

Tämä on mobiilipeli, jossa pelaaja voi tutkia luontoa ja kerätä löytöjä kartalta.

## Toiminta

Pelaaja liikkuu kartalla ja löytää erilaisia kohteita. Löydöt tallentuvat peliin ja niitä voi tarkastella myöhemmin.

## Ominaisuudet

- Karttanäkymä
- Löytöjen kerääminen kartalta
- Löytöjen tallennus
- Useamman luokan hyödyntäminen

## Toteutetut vaatimukset

**Perussuorituksesta toteutettu:**
- Sovelluksen perustoiminnallisuus
- Kartta ja interaktiot
- Useamman luokan käyttö

## Lisäominaisuudet

- Löydöt kartalla
- Useampi luokka sovelluksessa

## Tekniikat

- Android (Java/Kotlin)
